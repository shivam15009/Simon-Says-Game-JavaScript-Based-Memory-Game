# Simon Says Game – JavaScript Project  

A simple and interactive memory-based game built using HTML, CSS, and JavaScript. The game challenges users to remember and repeat an increasing sequence of colors.

---

## Features  

### Game Logic  
- Random color sequence generation  
- Increasing difficulty with each level  
- User input tracking and validation  

### Gameplay  
- Press any key to start the game  
- Follow the sequence shown by the system  
- Click buttons in the correct order  

### Scoring System  
- Level-based progression  
- High score tracking during gameplay  
- Game over message with score display  

### Visual Effects  
- Button flash effects for game and user input  
- Background color change on wrong input  
- Smooth transitions for better experience  

---

## Technology Stack  

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  

---

## Project Structure  

project-folder/
│── index.html

## How It Works

- The game starts when any key is pressed
- A random color is added to the sequence each level
- The user must repeat the exact sequence
- If the user makes a mistake, the game ends
- High score is updated automatically


## Run the Project

Open index.html in your browser
